

# תבניות תעודות (עורך ויזואלי) + מחירים להדרכות

## מה נבנה

### 1. מחיר בהדרכות
הוספת שדות תמחור לטופס יצירת הדרכה:
- **סוג תמחור**: לפי אדם / מחיר כולל
- **מחיר בסיס** (₪)
- **הנחה** (אחוזים, אופציונלי)
- **מחיר סופי**: מחושב אוטומטית (לפי אדם × מספר משתתפים, או גלובלי, פחות הנחה)

### 2. עורך תבניות תעודות (עמוד הגדרות)
עמוד `/settings` חדש עם עורך ויזואלי לתבניות תעודות:
- תצוגה מקדימה חיה (live preview) של התעודה בזמן עריכה
- שדות הגדרה: כותרת, טקסט גוף, צבעים, גופן, לוגו (טקסט בשלב זה), חתימה
- Placeholders שנכנסים אוטומטית: `{employeeName}`, `{companyName}`, `{trainingType}`, `{date}`, `{expiryDate}`, `{instructor}`, `{idNumber}`
- תבנית לכל קטגוריית הדרכה + תבנית ברירת מחדל
- העורך מציג את התעודה כ"דף" מעוצב עם אפשרות לשנות צבעי רקע, מסגרות, מיקום אלמנטים

### 3. צפייה בתעודה מעמוד התעודות
- כפתור "צפה" בכל שורה פותח דיאלוג עם התעודה המלאה מעוצבת
- כפתור "הורד PDF" דרך Print-to-PDF של הדפדפן

## פרטים טכניים

- העורך הויזואלי בנוי ב-React טהור (בלי ספריות drag-and-drop בשלב זה) — שדות טופס משמאל, תצוגה מקדימה מימין
- התבנית מאוחסנת כ-JSON עם הגדרות: צבע רקע, צבע מסגרת, צבע טקסט, גופן, טקסט כותרת, טקסט גוף עם placeholders
- הורדה כ-PDF דרך `window.print()` עם CSS מותאם להדפסה
- תבנית ברירת מחדל מוכנה מראש עם עיצוב מקצועי
