

# תעודות שעומדות לפוג — בחירת טווח זמן

## מה צריך
בדשבורד, במקום טווח קבוע של 3 חודשים, להוסיף בורר שמאפשר לבחור כמה חודשים קדימה לבדוק: 1, 2, או 3 חודשים.

## שינויים

### `src/pages/Index.tsx`
- הוספת state לטווח חודשים (ברירת מחדל: 3)
- 3 כפתורי toggle/select: "חודש", "חודשיים", "3 חודשים"
- סינון `expiringCerts` לפי הטווח שנבחר
- הצגת 5 תעודות בלבד (ממוינות לפי תאריך תפוגה הקרוב ביותר)
- כפתור ייצוא CSV עם כל התעודות בטווח שנבחר
- הצגת סה"כ תעודות שעומדות לפוג
