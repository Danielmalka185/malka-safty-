
# מערכת ניהול בטיחות – תוכנית פיתוח

## סקירה כללית
מערכת ניהול עסקית בעברית מלאה (RTL) לחברת בטיחות – ניהול חברות, עובדים, הדרכות, תעודות וסקרי סיכונים. עיצוב CRM מקצועי ומינימליסטי, מותאם לנייד.

## מבנה נתונים (Lovable Cloud / Supabase)
- **companies** – שם, ח.פ, איש קשר, טלפון, אימייל, כתובת, הערות
- **employees** – שם פרטי, שם משפחה, ת.ז, תאריך לידה, טלפון, תפקיד, חברה משויכת, סטטוס
- **training_types** – שם הדרכה, תחום, תוקף בחודשים, דורש תעודה
- **trainings** – חברה, סוג הדרכה, תאריך, מיקום, מדריך
- **training_participants** – טבלת קשר בין הדרכה לעובדים
- **certificates** – עובד, סוג הדרכה, הדרכה, תאריך הנפקה, תאריך תפוגה, סטטוס
- **risk_surveys** – שם אתר, חברה, תאריך, ממצאים, פעולות מתקנות
- **risk_survey_images** – תמונות מצורפות לסקר (Storage bucket)

## מסכים ותצוגות

### 1. דשבורד ראשי
- סטטיסטיקות: מספר חברות, עובדים פעילים, הדרכות החודש, תעודות שפג תוקפן
- כרטיסיות סיכום עם אייקונים
- רשימת תעודות שעומדות לפוג בקרוב
- גרף הדרכות לפי חודש

### 2. ניהול חברות
- טבלה עם חיפוש וסינון
- כרטיס חברה – פרטי חברה + טאבים: עובדים, הדרכות, סקרי סיכונים
- טופס הוספה/עריכה

### 3. ניהול עובדים
- טבלה עם חיפוש לפי שם/ת.ז/חברה
- כרטיס עובד – פרטים אישיים + רשימת הסמכות ותעודות
- סינון לפי סטטוס וחברה

### 4. סוגי הדרכות
- רשימה עם אפשרות הוספה ועריכה
- הגדרת תוקף ותחום

### 5. הדרכות (אירועים)
- יצירת הדרכה: בחירת חברה → סוג הדרכה → בחירת עובדים מרשימה (מולטי-סלקט)
- מילוי תאריך, מיקום, מדריך
- בסיום – יצירת תעודות אוטומטית לכל משתתף

### 6. תעודות
- טבלה עם חיפוש וסינון (לפי עובד, חברה, סטטוס, תאריך)
- חישוב אוטומטי של סטטוס (בתוקף/פג תוקף)
- כפתור הורדת PDF לכל תעודה

### 7. סקרי סיכונים
- טופס יצירה: אתר, חברה, תאריך, ממצאים (טקסט חופשי), פעולות מתקנות
- העלאת תמונות (Storage bucket)
- הפקת דוח PDF

## עיצוב ו-UX
- כיוון RTL מלא – כל הממשק בעברית
- סיידבר ניווט עם אייקונים
- עיצוב CRM נקי: צבעי כחול-אפור מקצועיים
- רספונסיבי מלא – תצוגת נייד מותאמת לעבודה מהשטח
- חיפוש מהיר גלובלי בראש המסך

## טכנולוגיה
- React + TypeScript + Tailwind (RTL)
- Lovable Cloud לבקאנד (DB, Auth, Storage, Edge Functions)
- PDF generation באמצעות Edge Function
- Storage bucket לתמונות סקרי סיכונים

## שלבי פיתוח
1. הגדרת RTL, עיצוב בסיסי, סיידבר וניווט
2. טבלאות DB + מסכי חברות ועובדים
3. סוגי הדרכות + מסך הדרכות עם בחירת משתתפים
4. תעודות – יצירה אוטומטית, תצוגה, הורדת PDF
5. סקרי סיכונים – טופס, העלאת תמונות, דוח PDF
6. דשבורד עם סטטיסטיקות
