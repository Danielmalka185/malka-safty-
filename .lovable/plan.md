

# ייבוא נתונים מ-Excel — עם קישור חברות-עובדים

## הבהרה חשובה
כל שורה בקובץ מכילה גם פרטי חברה וגם פרטי עובד. הסקריפט צריך:

1. **לחלץ חברות ייחודיות** — לפי שם מפעל (או ח.פ). חברה שמופיעה 20 פעם = חברה אחת.
2. **לחלץ עובדים ייחודיים** — לפי ת.ז. כל עובד מקבל `companyId` של החברה שהוא שייך אליה.
3. **קיבוץ הדרכות** — אותה חברה + אותו תאריך = הדרכה אחת עם כל המשתתפים.
4. **תעודות** — לכל עובד לפי ההסמכות המסומנות בשורה שלו.

## זרימת הקישור

```text
שורה ב-Excel:
  מפעל="נפטון פאתא" | ח.פ=514820362 | שם="יוסף" | ת.ז.=30572600 | תאריך=7/11/25 | סולמות=✓

  ↓ סקריפט מעבד:

  חברה { id: "c1", name: "נפטון פאתא", registrationNumber: "514820362" }
  עובד  { id: "e1", companyId: "c1", firstName: "יוסף", idNumber: "30572600" }
  הדרכה { id: "t1", companyId: "c1", participantIds: ["e1", ...], date: "2025-07-11" }
  תעודה { id: "cert1", employeeId: "e1", companyId: "c1", trainingTypeId: "1" }
```

## שלבי ביצוע

### 1. סקריפט Python
- קריאת הקובץ עם pandas
- חילוץ חברות ייחודיות → מילון `{שם_מפעל: company_id}`
- חילוץ עובדים ייחודיים (לפי ת.ז.) → שיוך `companyId` מהמילון
- קיבוץ הדרכות לפי חברה+תאריך → `participantIds` מכל העובדים באותה קבוצה
- יצירת תעודות לפי עמודות ההסמכה המסומנות
- ניקוי: ת.ז. (הסרת $/-), תאריכים (ISO), שדות ריקים

### 2. עדכון `src/data/mockData.ts`
- מחיקת כל הנתונים הקיימים (חברות/עובדים/הדרכות/תעודות לדוגמה)
- הוספת 6 סוגי הדרכה חדשים (גגות שבירים, סלים, קונסטרוקציה, חלל מוקף, פיגומים ממוכנים, גגות שטוחים)
- הכנסת הנתונים האמיתיים — עם קישורים נכונים בין כל הישויות

### 3. בדיקה
- הרצת בדיקה על חברה אחת (נפטון פאתא) — וידוא שכל 7 העובדים משויכים אליה, ההדרכה מכילה את כולם, והתעודות נכונות

|-------|--------|
| סקריפט Python (חד-פעמי) | ניתוח Excel, חילוץ וקישור נתונים |
| `src/data/mockData.ts` | החלפת נתוני דוגמה בנתונים אמיתיים מקושרים |
